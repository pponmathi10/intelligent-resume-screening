# Intelligent Resume Screening System

An AI-powered resume screening system using NLP and Machine Learning.

## Features
- Resume Upload (PDF)
- Skill Matching
- Match Score
- AI Recommendation
- Job Description Comparison
- Streamlit UI

## Tech Stack
- Python
- Streamlit
- Scikit-learn
- NLP (SpaCy)

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
